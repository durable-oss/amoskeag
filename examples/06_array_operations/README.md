# Array Operations

Working with arrays and collection functions.

## Demonstrates
- Array literals `[...]`
- Collection functions: `sum`, `avg`, `size`, `first`, `last`
- Dictionary literals `{...}`
- Combining multiple operations

## Expected Output
```json
{
  "total": 150,
  "average": 30,
  "count": 5,
  "first": 10,
  "last": 50
}
```

## Available Array Functions
- `sum` - sum all numbers in array
- `avg` - average of all numbers
- `size` - length of array
- `first` - first element (or nil)
- `last` - last element (or nil)
- `contains(val)` - check if array contains value
- `sort` - sort array
