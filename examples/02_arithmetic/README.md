# Arithmetic Operations

Basic mathematical operations in Amoskeag.

## Demonstrates
- Number literals
- Arithmetic operators: `+`, `-`, `*`, `/`, `%`
- Let bindings for intermediate calculations
- Order of operations

## Expected Output
```
123.0
```

Calculation: 100 + (100 * 0.08) + 15 = 100 + 8 + 15 = 123
